## DT50 Bluetooth Console: Privacy Policy

Diamond Technologies takes your privacy very seriously. 

To the best of our knowledge and belief, this application does not collect any personally identifiable information about the user or the device it is installed on.

## Required Permissions

This application discovers and connects to the Diamond Technologies DT50 Bluetooth Gateway.
In order to run properly, this application requires the following permissions:

 `android.permission.BLUETOOTH_ADMIN`
 </br>
 `android.permission.BLUETOOTH`
 </br>
 `android.permission.BLUETOOTH_CONNECT`
 </br>
 `android.permission.BLUETOOTH_SCAN`
 </br>
 `android.permission.ACCESS_FINE_LOCATION`
 </br>
 `android.permission.ACCESS_COARSE_LOCATION`
 
 </br>
 
| Permission | Why We Need to Ask |
| :---: | --- |
| `android.permission.BLUETOOTH`</br> | Allows the application to connect to DT50 Devices |
| `android.permission.BLUETOOTH_ADMIN`</br> | Allows the application to discover and pair with DT50 Devices |
| `android.permission.BLUETOOTH_CONNECT`</br> | Allows the application to connect to paired DT50 Devices |
| `android.permission.BLUETOOTH_SCAN`</br> | Allows the application to scan for nearby DT50 Devices |
| `android.permission.ACCESS_FINE_LOCATION`</br> | Allows the application to access the approximate device location. This permission is required to scan for Bluetooth Devices, as a Bluetooth Scan could possibly reveal information about the location of the user. This application does not store or transmit any location data. |
| `android.permission.ACCESS_COARSE_LOCATION`</br> | Allows the application to access the preceise device location. This permission is required to scan for Bluetooth Devices, as a Bluetooth Scan could possibly reveal information about the location of the user. This application does not store or transmit any location data. |
